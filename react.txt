useContext() allows react to consume values from React Context values ideal for managing
themes authentication and user preference. it prevents prop drilling
Prop drillling is when data is passed from parent to children and other nested elements that 
do not need it till it gets to the element that needs it

useEffect() is a react hook used to handle side effects in functional component eg fetching
data from an api, setting up event listeners, cleaning up a resource when it unmounts eg
useEffect(() => {code to run}
return() =>{clean up function(optional)}, [dependcies])
dependcies array if none runs after every render, 
[] if empty run once on mount and mimicks componentDidMount
[values, values]runs if there is a change in values mimicks  componentDidUpdate
return() inside the useEffect to clean up and mimick componentWillUnmount

useLayoutEffect() works like useEffect but rather works synchronously after the dom is done
loading meaning the web waits for it to compile before ui shows unlike useEffect which ui shows
before the effect is run we use it to adjust animation measure dom elements width & height
