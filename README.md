# 📚 Learning & Projects Repository

This repository is a central place for my **learning journey** in web development.  
It includes notes, exercises, and projects as I revise **Node.js fundamentals**, explore **advanced concepts**, practice **React + TypeScript**, and build both **mini projects** and larger applications.

---

##  Contents

### Fundamentals
- **Node.js Basics**: modules, events, streams, async programming
- **React Basics**: components, props, state, JSX
- **TypeScript**: interfaces, types, discriminated unions

### Advanced Topics
- Node.js: Express, middleware, APIs, authentication
- React: hooks (`useState`, `useReducer`, `useContext`), forms, reducers
- TypeScript: advanced typing, generics, payload handling

### Mini Projects
- Small exercises to reinforce concepts
- Practice with reducers, forms, and payloads
- Node revision projects (scripts, utilities)

### Bigger Projects (coming soon)
- Full-stack applications
- Organized monorepo-style structure
- Shared configs and documentation

---


## Key Concepts Practiced
- **State Management**: `useState`, `useReducer`
- **Payloads**: passing data with actions
- **Spread Operator (`...state`)**: preserving state while updating
- **Optional Chaining (`?.`) + Nullish Coalescing (`??`)**
- **Form Handling**: controlled inputs, button events
- **Node.js Fundamentals**: modules, async, event loop
- **Advanced Node.js**: APIs, middleware, error handling
